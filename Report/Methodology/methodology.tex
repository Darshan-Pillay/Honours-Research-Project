\section{Microservice architecture}
\begin{itemize}
	\item Use of dockerized containers allows reproducibility of our development environemnt on varying hardwares. Thus allowing computatinal repdoruciblity and other researchers to easily extend and reproduce our results with less effort \cite{boettigerIntroductionDockerReproducible2015}.
	\item A microservice architecture is a pattern of structuing a computing system application as a composition of microservices. A microservice is an independnet unit of the application which can be deployed run and scaled in isolation. Some benefits include modulatiry in that parts of the system can be replaced or extended without resambelling the whole system or having all the knowledge of the system. \cite{dragoniMicroservicesYesterdayToday2017, velepuchaSurveyMicroservicesArchitecture2023}
	\item Each service can be developed independelty and therefore choose the most suitable tool chains to solve the problem. There exsit numerous open-source software packages that can be used in plant phenotyping and the ability to implement a solution in any one of them and repalce them in the future is of great benefit to researchers. \cite{dragoniMicroservicesYesterdayToday2017} (benefits of microservice approach in this cite)
	\item Our image processing pipeline consists of a cluster of docker nodes running image processing services
	\item We use a dockerized ubuntu container with python installation as nodes
	\item Each node runs a flask rest API web service
	\item Reason for this choice is that we can implement on-site local network of nodes that dont rely on a cloud or central server (edge networks). Therefore keeping cost low \cite{grecoComputerVisionAlgorithms2024}

\subsection{Preprocessing Phase}
\begin{itemize}
	\item Color conversion and histogram ewulaiztion used to imporve the quality and clarity of of images. \cite{thangaduraiComputerVisionimageEnhancement2014}. Histogram equalization enhances the contrast
	\item Hue channel is best for segmentation it cast all intenstiy related infromation \cite{ahmedLeafImagebasedPlant2021}
	\item Histogram equalization used already for preprocessing \cite{arivazhaganDetectionUnhealthyRegion2013}
	\item Plants show relevaltivly low reflectance in the red and blue regions of the spectrum with peak in the green band. Non-normalized RGG values are directly propotional to the total light reflected on the surface therefore they are highly senstive to intensity of illuminating source and viewing angles \cite{perezColourShapeAnalysis2000}. Can apply NDI index to reduce lighting problems
	\item Preprocessing involves removing noise, and enhancing the image so that salient features of the image are analysed  more effectivley in downstream phases of the pipeline. For example adjusting contrast mitigates varying lighting, shadows, and lumincance issues which can affect the overall accuracy of segmentation.colour conversion space is used to address lighting issues. and filtering also plays important role in image enhancement and has effectively been used for plant leaf diesaes detection. \cite{hamudaSurveyImageProcessing2016a} (see the preprocessing phase section for more citations regarding the filtering, and image enhancement)
	\item Color Space Conversion ino HSV (removes luminance issues so more robust to varying lighting conditions). \cite{kimDevelopmentLowcostAutomated2024, wangColorImageSegmentation2003}. This color space mimics or tries to reprent the aritists notions of hue, tint, shade and tone \cite{smithColorGamutTransform1978}. HSV is more intuitive for human perception. And robust to lighting (illumination changes). It offers superior contrast and visial clarity ti distinguis between healthy and diseased regions in leaf images. It can enhance visibility of subtle disease patterns \cite{hussainRobustLeafDisease2024}. HS channels not affected by light changes and can therefore express the hue and brightness of colors to facilate color contrast \cite{fanEstimatingAquaticplantArea2021}. The images in the plantvillage dataset are set against a grey or black background, and noise from camera flash and varying illumination will affect segmentation. A tomotoe leaf whether diseased or healthy has color properties significantly different from the background, so based on this assumption we are justified in this case in using the hue channel to segement our image.
	\item use guassian filters to remove noise and unncessary information from image.
	\item the histrograme equalization tehcnique is used to adjust the contrast of the image.\cite{saiImageQualityEnhancement2023}
	\item convert image to gray scale image
\end{itemize}

\subsection{Segmentation Phase}
\begin{itemize}
	\item Use this resource to find brief sentence describing otsu method \cite{perez-sanzPlantPhenomicsOverview2017}
	\item Otsu method is a widely applied threshold segmentation method in plant phenotyping. It's primary advantages include simple calculation and high efficiency making it very attrative for low-cost image processing pipelines. It is however sensitive to noise, and has low robustness to varying light conditions where bimodal histogram distrubutions are not possible. However emperically it has been succesfully in the phenotyping of many different plant organs across different species \cite{liReviewComputerVision2020} (you can include other sources from this article look at the table summarizing the 2d segmentation methods and where otsu is used). It is a global threshold and therefore has simple implementation. and stable performance and is widley used in plant image segmentation \cite{fanEstimatingAquaticplantArea2021}. The otsu method selects best threshold on on maximum interclass variance. Suitable for bimodal distributions, but where unimodal histrogram or non-bimodal performance is poort. \cite{fanEstimatingAquaticplantArea2021}. We apply preprocessing steps which attempt to normalize the histogram and and mitigate lumincance issues to increase the effectivness of our segmentation.
	\item We apply otsu segmentation thresholding method
	\item due to varying lighting conditions across all images we use adaptive thresholding
\end{itemize}

\section{Classification  Mechanism}
\begin{itemize}
	\item The gray level co-occurence methodlogyu is a statisitcal way to descibe the shape by smapling the way gray levels ocuren in relation to other gray levels. And have been applied successfully to detect and clasify plant pathogens \cite{arivazhaganDetectionUnhealthyRegion2013, mokhtarSVMBasedDetectionTomato2015}.
	\item We pass feature vector calculated from gray level co-occurence matrix
	\item This feature vector is fed into SVM classifier which determines whether or not original image leaf has disease or not

	\item Use simple dataset split or 10-fold cross validation
